# Covid-19 Simulation (Python)

**License:** MIT‚ÄÉ‚ÄÉ**Language:** Python‚ÄÉ‚ÄÉ**Framework:** Streamlit‚ÄÉ‚ÄÉ**Type:** Markov Simulation  

Simulates the spread of COVID-19 using a per-country sampled population and a simple Markov-chain per-person model.  
This project generates individual and country-level time-series data and visualizes infection waves through plots and a Streamlit web app.

---

## üß† Overview

This project models the spread and recovery of COVID-19 in different countries using a probabilistic simulation.  
It operates on a per-person daily simulation model, where each person transitions between different health states based on predefined probabilities.  

The project provides:
- A **detailed daily simulation** for each individual in the sample population.  
- **Aggregated summaries** of infection trends per country.  
- A **visual plot** of infection waves across time.  
- An optional **Streamlit UI** to interactively view and run simulations.

---

## üìä Health States (Markov Chain Model)
Each simulated person transitions daily between the following states:
- **H:** Healthy  
- **I:** Infected (no symptoms)  
- **S:** Symptomatic  
- **M:** Immune (recovered)  
- **D:** Deceased  

Transition probabilities and durations are defined in `sim_parameters.py`.

## üß© Key Files

| File | Description |
|------|--------------|
| `assignment2.py` | Main simulator ‚Äî implements `run()` for running the full simulation. |
| `app.py` | Streamlit web app for interactive visualization. |
| `helper.py` | Contains plotting helper (`create_plot()`) to generate PNG output. |
| `sim_parameters.py` | Defines transition probabilities and holding times per age group. |
| `test.py` | Driver script that runs the simulation with sample inputs. |
| `a2-countries.csv` | Input data for country demographics. |

---

## üì¶ Outputs

Running the simulation generates the following files:
- `a2-covid-simulated-timeseries.csv` ‚Äî Detailed per-person/day/state simulation results.  
- `a2-covid-summary-timeseries.csv` ‚Äî Aggregated counts per country/day for each state (H, I, S, M, D).  
- `a2-covid-simulation.png` ‚Äî Visualization plot created by `helper.py::create_plot()`.
